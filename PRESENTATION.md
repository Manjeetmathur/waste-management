## Slide 11: User Journey - Scheduling Pickup

### Step-by-Step Process

1. **Sign Up/Sign In**
   - Create account or use Google Sign-In
   - Profile setup

2. **Schedule Pickup**
   - Select waste type (Plastic, Paper, Metal, etc.)
   - Enter weight estimate
   - Add address and preferred date/time
   - Upload waste images (optional)

3. **Get Matched**
   - System matches with nearby recyclers
   - See price estimate
   - Confirm details

4. **Pickup & Payment**
   - Recycler collects waste
   - Payment processing
   - Earn points
   - Rate experience
